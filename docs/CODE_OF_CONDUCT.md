# Contributor Covenant 行为准则

## 我们的承诺

作为成员、贡献者与维护者，我们承诺将社区参与体验建设为一个无骚扰环境。无论年龄、体型、显性或隐性残障、族裔、性特征、性别认同与表达、经验水平、教育背景、社会经济地位、国籍、外貌、种族、种姓、肤色、宗教或性取向如何，所有人都应获得尊重。

我们承诺以开放、欢迎、多元、包容且健康的方式进行协作与互动。

## 我们的标准

有助于营造积极社区环境的行为包括：

- 对他人展现同理心与善意
- 尊重不同观点、经验与立场
- 提供建设性反馈，也能坦然接受建设性反馈
- 对自己的错误负责，并向受影响的人致歉，从中学习
- 关注整个社区的长期利益，而不仅仅是个人利益

不可接受的行为包括：

- 使用带有性暗示的语言或图像，或任何形式的不当性关注与性挑逗
- 侮辱、贬损、钓鱼、嘲讽，以及人身或政治攻击
- 公开或私下骚扰
- 未经明确许可公开他人的隐私信息，例如住址或邮箱
- 其他在专业协作环境中可被合理认定为不当的行为

## 执行责任

社区维护者有责任阐明并执行可接受行为的标准，并对任何他们认定为不当、威胁、冒犯或有害的行为采取公平且适当的纠正措施。

维护者有权移除、编辑或拒绝不符合本行为准则的评论、代码、提交、Wiki 编辑、Issue 及其他贡献，并在适当情况下说明处理原因。

## 适用范围

本行为准则适用于所有社区空间，也适用于个人在公开场合代表社区的情形。包括但不限于：使用官方邮箱、通过官方社交媒体发声，或在线上 / 线下活动中以社区代表身份参与。

## 举报与执行

若遇到辱骂、骚扰或其他不可接受行为，请发送邮件至 **sam@harnetics.dev**。

所有投诉都会被及时且公平地审查与调查。

所有负责执行本准则的维护者都有义务尊重举报人的隐私与安全。

## 执行指南

社区维护者会参考以下“社区影响等级”来决定违规处理方式：

### 1. 纠正

**社区影响**：使用了不恰当语言，或出现其他被认定为不专业、令人不适的行为。

**处理结果**：维护者会发出私下书面提醒，说明违规性质及其不当原因；必要时可要求公开道歉。

### 2. 警告

**社区影响**：发生单次或连续多次违规行为。

**处理结果**：发出正式警告；若继续违规，将承担进一步后果。在指定期限内，不得与相关当事人互动，也不得主动联系执行本准则的人员；这同样适用于社区空间及外部渠道（如社交媒体）。违反这些要求可能导致临时或永久封禁。

### 3. 临时封禁

**社区影响**：严重违反社区标准，包括持续性不当行为。

**处理结果**：在指定期限内，禁止与社区发生任何公开或私下互动。包括但不限于：不得与相关当事人接触，也不得主动联系执行本准则的人员。违反这些要求可能导致永久封禁。

### 4. 永久封禁

**社区影响**：形成持续违规模式，包括长期不当行为、针对个人的骚扰，或对特定群体的攻击与贬损。

**处理结果**：永久禁止参与社区中的任何公开互动。

# Reliability

当前没有运行中的应用，所以这里记录的是“准备好可运行系统之前必须守住的可靠性底线”。

现阶段校验点：
- `fixtures/` 中每份文档都必须有稳定 `doc_id`、版本和来源关系
- 迁移或重命名文档时，要同步更新引用路径与 `AGENTS.md`
- 新增执行计划前，先确认它挂在 `docs/exec-plans/active/`

未来服务化后的第一批健康检查应至少包含：
- 文档导入是否成功解析为结构化 section
- ICD 参数是否能建立版本差异
- 草稿生成是否保留来源引用
- 影响分析是否能输出受影响文档清单

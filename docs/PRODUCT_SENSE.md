# Product Sense

Harnetics 不是通用写作助手，而是商业航天场景里的文档对齐工具。

核心用户：
- 分系统工程师：想更快得到带引用的文档初稿
- 技术负责人：想知道上游文档变更会影响谁
- 部署管理员：想在本地、安全、低门槛地把系统跑起来

核心价值：
- 把 2 到 3 天的“找文档 + 对齐引用”压缩到半天以内
- 让引用、版本、接口参数成为可见结构，而不是靠邮件和记忆传递

产品边界：
- 先解决“跨部门文档对齐”
- 暂不扩张到完整 PLM、需求管理或全企业知识平台

# New User Onboarding

目标：让第一次接触系统的工程师在最短路径内完成“导入 -> 检索 -> 生成草稿”。

建议首登流程：
1. 管理员导入一组样本文档或真实受控文档。
2. 工程师选择部门、文档类型与目标主题。
3. 系统展示候选上游文档与模板。
4. 工程师点击生成，拿到带引注的草稿。
5. 系统同时显示引用来源与潜在版本冲突。

这份文档目前是规格占位页；等第一轮交互方案定稿后，应补充页面状态、异常路径与成功判定。

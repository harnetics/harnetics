# Core Beliefs

- 文档不是附件，文档就是数据本身；系统价值来自让这些数据可追溯、可对齐、可复用。
- 先解决一个真实痛点，不做抽象完美的“全流程平台”。
- 引用必须可见，版本必须可见，冲突必须可见；隐藏这些结构只会把复杂度转移给工程师。
- 本地优先不是部署偏好，而是信任边界。
- 未来代码应围绕一个单一闭环生长：导入 -> 建图 -> 生成 -> 校验 -> 影响分析。

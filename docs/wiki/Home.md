# Harnetics Wiki

Harnetics 是一个**商业航天文档对齐工作台**，面向航天工程团队的跨部门追溯、草稿生成与变更影响分析。

## 导航

- [快速开始](Quick-Start.md)
- [架构总览](Architecture-Overview.md)
- [贡献入口](Contributor-Guide.md)
- [术语与场景](Glossary-and-Use-Cases.md)

## 当前重点能力

1. 文档追溯：把来源文档、章节与引用关系落到统一图谱中
2. 草稿生成：基于图谱上下文生成带引注的 Markdown 草稿
3. 影响分析：在文档变更后沿引用图分析波及范围

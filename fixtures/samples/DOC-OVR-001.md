# 天行一号运载火箭总体设计方案

## 1. 概述

### 1.1 型号概况

天行一号（TH-1）运载火箭是天行航天科技有限公司研制的中型两级液体运载火箭，主要面向近地轨道（LEO）和太阳同步轨道（SSO）中小型卫星发射市场。

**主要技术指标**：
- 运载能力：1,500 kg @ 500 km SSO（参见 DOC-SYS-001 REQ-SYS-001）
- 全箭高度：42.5 m
- 一级直径：3.35 m
- 发射质量：约 180 t
- 推进剂：液氧/液甲烷（LOX/LCH4）

### 1.2 引用文件

- DOC-SYS-001 天行一号运载火箭系统级需求文档 v3.1
- DOC-ICD-001 天行一号全局接口控制文档 v2.3
- DOC-DES-001 TQ-12 液氧甲烷发动机分系统设计文档 v1.2

## 3. 飞行程序

### 3.1 主要时序节点

| 时间 (s) | 事件                    |
| -------- | ----------------------- |
| T-3.0    | 一级发动机点火         |
| T+0.0    | 起飞（推重比 > 1.2）   |
| T+10.0   | 程序转弯开始           |
| T+60.0   | Max-Q（最大动压）      |
| T+120.0  | 一级变推力段开始       |
| T+185.0  | 一级发动机关机（MECO） |
| T+188.0  | 一二级分离             |
| T+190.0  | 二级发动机点火         |
| T+500.0  | 整流罩抛罩             |
| T+600.0  | 二级发动机关机（SECO） |
| T+620.0  | 星箭分离               |

### 3.2 一级动力参数

- 发动机型号：TQ-12 液氧甲烷发动机
- 地面推力：650 kN（DOC-ICD-001 ICD-PRP-001）
- 真空推力：720 kN
- 工作时间：185 s（DOC-ICD-001 ICD-PRP-009）
- 启动过程：T-3.0 s 点火，T+0.0 s 达到 90% 推力（启动时间 ≤ 3.0 s，REQ-SYS-008）

## 4. 分系统设计指标

### 4.1 箭体结构

- 一级壳体材料：2219 铝合金
- 贮箱构型：共底贮箱（LOX 在上、LCH4 在下）
- 结构质量比：6.7% （满足 REQ-SYS-011 ≤ 7%）

### 4.2 一级动力系统

- 发动机型号：TQ-12
- 数量：1 台
- 推力：650 kN（地面），720 kN（真空）
- 比冲：316.5 s（真空）（DOC-DES-001 §2.1）
- 混合比：3.5:1（DOC-ICD-001 ICD-PRP-003）
- 发动机干重：470 kg（DOC-DES-001 §2.1）
- 推力调节范围：50%~100%（DOC-ICD-001 ICD-PRP-012）

> 注：上述参数均以 DOC-ICD-001 v2.3 为准。

### 4.3 二级动力系统

- 发动机型号：TQ-8
- 数量：1 台
- 真空推力：80 kN
- 真空比冲：345 s
- 多次启动：≥ 3 次

### 4.4 GNC 系统

- 惯性导航：光纤陀螺 + 石英加速度计（IMU 三冗余）
- 制导律：迭代制导
- 姿态控制：发动机摇摆（±6°）+ 冷气 RCS
- 控制精度：±0.5°（满足 REQ-SYS-010）

### 4.5 测控系统

- 遥测频段：S 波段
- 遥测速率：2 Mbps
- 外弹道测量：GPS + 雷达
- 安控系统：双冗余接收 + 双爆炸螺栓

---

## 5. 地面系统

### 5.1 发射台

- 类型：自行式发射台
- 供应管路：LOX（DN150）、LCH4（DN100）、高压氮气（DN50）

### 5.2 测试发射控制中心

- 距发射台：2.5 km
- 数据通道：光纤 + 无线备份
- 控制方式：自动化发射流程 + 人工确认节点

## 6. 技术风险

| 风险项             | 级别 | 缓解措施                     |
| ------------------ | ---- | ---------------------------- |
| TQ-12 推力偏低     | 中   | 预留推力裕度，涡轮泵优化     |
| 共底贮箱密封       | 高   | 地面低温试验验证             |
| 一二级分离冲击     | 中   | 仿真分析 + 分离试验          |
| Max-Q 气动载荷     | 低   | CFD 仿真 + 风洞试验          |

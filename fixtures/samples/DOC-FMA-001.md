# TQ-12 动力系统故障模式与影响分析（FMEA）

## 1. 概述

### 1.1 目的

本文档对 TQ-12 液氧甲烷发动机的故障模式进行系统性识别和分析，评估各故障模式对分系统和全系统的影响，确定关键项目和风险缓解措施。

### 1.2 分析范围

本次分析覆盖 TQ-12 发动机以下主要组件：
- 推力室组件（含喷注器、燃烧室、喷管）
- 涡轮泵组件（LOX 泵、LCH4 泵）
- 预燃室
- 阀门系统
- 控制器
- 管路与连接件

### 1.3 引用文件

- DOC-SYS-001 天行一号运载火箭系统级需求文档 v3.1
- DOC-ICD-001 天行一号全局接口控制文档 v2.3
- DOC-DES-001 TQ-12 液氧甲烷发动机分系统设计文档 v1.2
- DOC-QAP-001 天行一号质量管理计划 v1.0
- GJB 1391A 故障模式、影响及危害性分析

### 1.4 严重度分级

| 级别 | 名称   | 定义                                   | 等级 |
| ---- | ------ | -------------------------------------- | ---- |
| I    | 灾难   | 导致人员伤亡或运载器损毁               | 1    |
| II   | 致命   | 导致任务失败                           | 2    |
| III  | 临界   | 导致任务降级或性能下降                 | 3    |
| IV   | 轻微   | 不影响任务完成，可在维护中解决         | 4    |

### 1.5 发生概率分级

| 级别 | 名称   | 概率范围           |
| ---- | ------ | ------------------ |
| A    | 频繁   | > 10⁻¹            |
| B    | 很可能 | 10⁻¹ ~ 10⁻²      |
| C    | 偶然   | 10⁻² ~ 10⁻³      |
| D    | 远端   | 10⁻³ ~ 10⁻⁶      |
| E    | 极不可能 | < 10⁻⁶          |

## 4. 关键项目清单

### 4.1 单点故障项

以下故障模式被识别为可能导致灾难性后果的单点故障：

| FM编号    | 故障模式               | 缓解措施状态   | 残余风险 |
| --------- | ---------------------- | -------------- | -------- |
| FM-TC-001 | 燃烧室过压             | ✅ 安全阀+关机 | 低       |
| FM-TC-003 | 高频不稳定             | ✅ 声学设计    | 中       |
| FM-TC-004 | 冷却通道堵塞           | ⚠️ 依赖工艺控制 | 中     |
| FM-TP-002 | 轴承卡死               | ✅ 冗余设计    | 低       |
| FM-VL-001 | 主阀卡开               | ✅ 冗余关机    | 低       |

### 4.2 风险矩阵

```
         │ 灾难(I) │ 致命(II)│ 临界(III)│ 轻微(IV)
─────────┼─────────┼─────────┼──────────┼─────────
频繁(A)  │         │         │          │
很可能(B)│         │         │          │
偶然(C)  │ TP-002  │         │ TC-002   │ CT-001
         │ TP-003  │         │ TC-005   │
         │ VL-004  │         │ TP-004   │
         │         │         │ VL-003   │
         │         │         │ CT-003   │
远端(D)  │ TC-001  │ TC-006  │          │
         │ TC-003  │ PB-002  │          │
         │ TC-004  │ VL-002  │          │
         │ TP-001  │ CT-004  │          │
         │ TP-005  │         │          │
         │ PB-001  │         │          │
         │ VL-001  │         │          │
极不可能(E)│CT-002│         │          │
```

**高风险区域**（需优先关注）：严重度 I + 发生概率 C 的项目
- FM-TP-002 轴承磨损/卡死
- FM-TP-003 密封泄漏
- FM-VL-004 阀门外泄漏

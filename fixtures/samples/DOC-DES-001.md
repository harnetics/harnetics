# TQ-12 液氧甲烷发动机分系统设计文档

## 1. 文档说明

### 1.1 目的

本文档描述天行一号运载火箭一级主发动机 TQ-12 的分系统设计方案，包括总体设计参数、各组件设计、接口定义及设计裕度分析。

### 1.2 引用文件

- DOC-SYS-001 天行一号运载火箭系统级需求文档 v3.1
- DOC-ICD-001 天行一号全局接口控制文档 v2.3
- DOC-FMA-001 TQ-12 动力系统故障模式与影响分析 v2.0

### 1.3 设计约束

- 推进剂：液氧（LOX）+ 液甲烷（LCH4）
- 循环方式：分级燃烧循环（富燃预燃室）
- 设计寿命：单台发动机累计试车时间 ≥ 2000 s 或 20 次启动
- 可复用性：设计考虑未来复用需求，关键组件设计寿命 ≥ 10 次飞行

## 3. 组件设计

### 3.1 推力室

- 推力室材料：铜合金（CuCrZr）内壁 + 镍基高温合金外壁
- 冷却方式：再生冷却，液甲烷作为冷却剂
- 喷管面积比：40:1（地面版）
- 喷注器类型：同轴剪切式喷注器，120 对喷注孔
- 点火方式：火炬点火器（TEA-TEB 化学点火）

### 3.2 涡轮泵

- LOX 涡轮泵转速：18,000 rpm
- LCH4 涡轮泵转速：25,000 rpm
- 泵效率：≥ 70%
- 轴承类型：液态氧润滑的静压轴承
- 密封方式：浮动环密封 + 气膜密封

### 3.3 预燃室

- 类型：富燃预燃室（fuel-rich staged combustion）
- 预燃室压力：28 MPa
- 预燃室温度：800 K
- 预燃混合比：0.4:1（LOX:LCH4）

### 3.4 阀门系统

- 主阀：电液伺服阀（推力调节用），响应时间 < 50 ms
- 启动/关机阀：气动球阀，动作时间 < 200 ms
- 安全阀：弹簧式安全阀，开启压力 12 MPa

### 3.5 控制器

- 处理器：三冗余 FPGA 控制器（TMR 架构）
- 通信接口：MIL-STD-1553B 双冗余总线（DOC-ICD-001 ICD-PRP-007）
- 传感器通道：32 路模拟量（压力、温度、流量）+ 16 路数字量
- 采样速率：1 kHz（关键参数）/ 100 Hz（一般参数）
- 控制算法：PID + 前馈补偿

## 4. 接口设计

### 4.1 机械接口

按照 DOC-ICD-001 ICD-PRP-006 定义，发动机安装接口采用法兰螺栓连接：
- 法兰直径：Φ420 mm
- 紧固件：24×M16 高强度螺栓，12.9 级
- 法兰平面度：≤ 0.1 mm
- 安装角度精度：≤ 0.05°

### 4.2 推进剂接口

按照 DOC-ICD-001 ICD-PRP-008 定义：
- LOX 入口：DN80 法兰连接，入口压力 ≥ 0.8 MPa
- LCH4 入口：DN65 法兰连接，入口压力 ≥ 0.6 MPa
- 预冷排放口：DN25，排放至安全排放系统

### 4.3 电气接口

按照 DOC-ICD-001 ICD-PRP-007 定义：
- 控制总线：MIL-STD-1553B，连接器 MIL-DTL-38999 Series III
- 电源接口：28V DC ±4V，峰值功率 500W
- 点火指令接口：离散量，光电隔离

## 5. 故障模式与安全设计

### 5.1 关键故障模式

参考 DOC-FMA-001，TQ-12 已识别的关键故障模式包括：

| 故障模式              | 严重度 | 发生概率 | 检测方式           | 缓解措施            |
| --------------------- | ------ | -------- | ------------------ | ------------------- |
| 燃烧室压力异常升高    | 灾难   | 远端     | 压力传感器实时监测 | 紧急关机，安全阀    |
| 涡轮泵轴承失效        | 灾难   | 偶然     | 振动传感器         | 冗余轴承设计        |
| 推进剂泄漏            | 灾难   | 偶然     | 泄漏检测器         | 紧急关机            |
| 喷注器堵塞            | 致命   | 偶然     | 混合比异常检测     | 滤网 + 清洗工艺     |
| 控制器单机故障        | 致命   | 偶然     | TMR 表决           | 三冗余设计          |

### 5.2 紧急关机逻辑

| 参数           | 告警阈值     | 关机阈值       | 响应时间 |
| -------------- | ------------ | -------------- | -------- |
| 燃烧室压力     | > 10.5 MPa   | > 11.0 MPa     | < 50 ms  |
| 涡轮入口温度   | > 850 K      | > 900 K        | < 100 ms |
| 推力/混合比偏差 | > ±8%       | > ±12%         | < 100 ms |
| 振动量级       | > 15 g       | > 25 g         | < 50 ms  |
